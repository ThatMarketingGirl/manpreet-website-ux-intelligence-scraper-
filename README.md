# Website UX Intelligence Analyzer

This Skill analyzes any public website and converts it into structured UX and design intelligence.

It does NOT clone websites or extract assets. It focuses on:
- UX structure breakdown
- Motion and animation patterns (descriptive only)
- Design system inference
- Conversion flow analysis
- Rebuild guidance

---

## What it does

When you provide a URL, it:

1. Asks clarifying questions
2. Fetches visible HTML content
3. Extracts structure and patterns
4. Analyzes UX and motion behavior
5. Outputs a structured inspiration report

---

## Output includes

- Page structure
- UX flow
- Interaction patterns
- Motion behavior description
- Design system inference
- Rebuild suggestions

---

## Important limitation

This tool does NOT:
- clone UI
- download assets
- replicate designs
- extract proprietary code

It is for inspiration and analysis only.
