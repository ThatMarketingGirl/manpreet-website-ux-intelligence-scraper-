# Website UX Intelligence + Figma Wireframe Generator

This Skill analyzes any public website and converts it into:

- UX structure breakdown
- Visual hierarchy analysis
- Motion & interaction patterns (descriptive only)
- Design system inference
- Figma-style wireframe blueprint

It does NOT clone websites or copy assets.
